## Project Overview

This repository contains my Digital Literacy Portfolio, created as part of the Digital Literacy course at VIT Bhopal University. In this project, I played the role of a and completed five tasks to help my peers navigate the digital world responsibly.

The portfolio covers:
1. Creating awareness about digital literacy
2. Building professional online profiles
3. Exploring coding and collaboration platforms
4. Practicing email etiquette
5. Understanding cybercrime and prevention

## Tasks Completed

### Task 1: Digital Literacy Awareness Infographic
Created a one-page infographic using **Canva** covering:
1. What is digital literacy
2. Useful tools for students (Google Workspace, Notion, OneNote)
3. Email etiquette tips

### Task 2: Student Digital Portfolio
Set up professional profiles on:
1. **GitHub** – for hosting code and projects
2. **LinkedIn** – for professional networking
3. **Kaggle** – for data science practice

### Task 3: Coding & Collaboration Platforms
1. Completed a beginner coding challenge on **HackerRank**
2. Created a **Google Form** with 5 questions for Digital Literacy Awareness Quiz


### Task 4: Email Etiquette & Social Media Checklist
1. Drafted two professional emails (to professor and internship coordinator)
2. Created a Social Media Do's and Don'ts checklist with 5 points each


### Task 5: Cybercrime Awareness
1. Wrote a case study on **phishing** with a fictional scenario
2. Created a "Stay Safe Online" prevention checklist with 8 tips
3. Included reporting channels: cybercrime.gov.in and helpline 1930
